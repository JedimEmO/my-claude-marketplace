# dwind

Skills for building UIs with the dwind/dominator/futures-signals Rust+WASM stack.

## Skills

- **dwind-project-setup** — Project creation, Rust-to-WASM build pipeline, wasm-pack config
- **dwind-component** — Component creation, #[component] macro, dwui widgets
- **dwind-styling** — CSS classes, layout, responsive design, hover/focus states, theming
- **dwind-reactivity** — Signals, Mutable, reactive updates, conditional rendering
- **dwind-design-system** — Design tokens, spacing/type scales, color systems, accessibility
- **dwind-tauri** — Tauri desktop apps with dwind/dominator frontend
