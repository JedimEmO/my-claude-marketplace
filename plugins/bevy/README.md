# Bevy

Skills for building games and interactive applications with the Bevy ECS game engine in Rust.

## Skills

- **bevy-project-setup** — Project scaffolding, Cargo.toml, compile times, feature flags
- **bevy-ecs** — Entity Component System, queries, commands, resources, events, system ordering
- **bevy-rendering** — 2D/3D rendering, sprites, meshes, materials, cameras, lighting, shaders
- **bevy-physics** — Collision detection, rigid bodies, raycasting, character controllers
- **bevy-input-and-interaction** — Keyboard, mouse, gamepad, touch input, picking/raycasting
- **bevy-ui-and-audio** — Game menus, HUD, text display, audio playback, spatial audio
- **bevy-ecosystem** — Third-party crates, version compatibility, migration guidance
