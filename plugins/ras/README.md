# RAS (Rust Agent Stack)

Skills for building type-safe services with the Rust Agent Stack — REST, JSON-RPC, file, and WebSocket APIs via macro-driven code generation.

## Skills

- **ras-setup** — Project creation, workspace config, dependency management
- **ras-api-design** — REST endpoints, JSON-RPC methods, macro syntax, OpenAPI generation
- **ras-security** — Authentication, authorization, JWT, OAuth2, RBAC, permission guards
- **ras-best-practices** — Observability, error handling, metrics, service-to-service communication
